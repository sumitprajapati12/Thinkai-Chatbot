# Thinkai-Chatbot

# Description
It is a AI Chatbot that can interact with users.
The Chatbot will understand simple messages like "hello", "motivate me" and respond with friendly, intelligent answers.
This is a "rule-based chatbot", it doesn't use real AI models yet, but it behaves intelligently using python logic

# Features
1) Responds to simple user inputs
2) Works continuously until the user type "bye"
